# 食材管理サイト "SYOKUKAN"のサイトです

###使用手順
1. まずは、TOPサイトにて買った食材とその期限を入力しておきます
2. そうすると、食材がリスト化されて表示されるので、もし変更などが必要であれば編集する
3. 期限が近かったり、期限切れの場合は、食材のタブの色が変化するので、なるべく早く食べなければいけないという合図になります

###今後の改善
今はReactで行っていますが、今後別機能を用いてオンライン上でデータ共有をしたり、AI機能を搭載するなどして、改善を行っていく予定です
